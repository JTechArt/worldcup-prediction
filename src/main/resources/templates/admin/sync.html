Individual Jobs

Each job respects the rate limiter (10 calls/min). Skipped jobs return instantly when there's nothing to do.

Teams + Squads

Update 48 teams with API external IDs + populate full squads (players table). 1 API call.

Group Stage Matches

Clear wrong seed data and recreate 72 group stage matches from API with correct groups + kickoff times. 1 API call.

Standings

Upsert group standings from API. Skipped if no matches completed since last sync. 1 API call.

Lineups + Goals

Fetch lineup and goal data for completed matches that haven't been enriched yet. 1 API call per match.

Top Scorers

Update player tournament goal counts. Skipped if no matches completed in last 24h. 1 API call.
